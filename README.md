## Unity version
- 6000.1.10f

## 빌드 및 실행 방법
- 해당 프로젝트를 git clone하여 받고, 폴더를 unity hub에서 디스크에서 추가를 통해 불러온다.
- Build Platform을 Android로 설정하고 메타 퀘스트가 컴퓨터에 연결된 상태에서 Build and run을 실행한다.
- 또는 Build를 수행하여 apk 파일을 얻어내고, Meta Quest Developer Hub를 통해 해당 파일을 Meta Quest에 설치한다.
- 해당 프로젝트 실행을 위해서는 공간 인식 및 가구 설정이 필요하고, 반드시 "벽에 걸린 그림"이 구조에 추가되어야 한다.
- 집에 벽에 걸린 그림이 없다면 벽 오브젝트에 임의의 사각형 오브젝트를 생성하고 "벽에 걸린 그림"으로 설정한다.
